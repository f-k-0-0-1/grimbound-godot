# Grimbound

> A handcrafted 2D action-platformer built with **Godot 4**, featuring fast-paced combat, exploration, hidden secrets, and challenging platforming.

> **Status:** 🚧 In Development

---

## About

**Grimbound** is a single-player 2D platformer where you play as a mysterious hooded warrior bound by an ancient curse. Traverse forgotten ruins, uncover hidden relics, defeat dangerous enemies, and restore what has been lost.

The project is being developed with a strong focus on clean architecture, modular systems, and maintainable code using Godot's best practices.

---

## Features

### Gameplay

- ⚔️ Responsive melee combat
- 🏃 Smooth platforming movement
- 🦴 Multiple enemy types
- 👑 Boss battles
- 🗝️ Hidden areas and secrets
- 💎 Collectibles and treasures
- ❤️ Health and checkpoint system
- 🎯 Progressive level design

### Technical

- Modular project architecture
- Scene inheritance
- Resource-driven data (`.tres`)
- State machine based AI
- Signal-driven communication
- Android-first optimization
- Clean folder structure
- Reusable game systems
